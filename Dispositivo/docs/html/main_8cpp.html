Detailed Description

Programa principal para simular el uso de dispositivos inteligentes.

Este programa permite registrar varios dispositivos inteligentes, mostrar su información, simular su consumo energético en función del tiempo de uso y controlar el acceso remoto mediante bloqueo y desbloqueo.

Cada dispositivo se representa con la clase DispositivoInteligente, que administra su nombre, estado (encendido/apagado), consumo energético y si su control remoto está bloqueado o no.
