Detailed Description

Clase que modela un dispositivo inteligente con control de energía y estado.

Esta clase proporciona funcionalidades para manipular el estado de un dispositivo, conocer su consumo energético y controlar su acceso remoto mediante bloqueo o desbloqueo.
